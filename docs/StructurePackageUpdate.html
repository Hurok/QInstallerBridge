struct QInstallerBridge::PackageUpdate

This struct is used all over QInstallerBridge to store the information on new updates. Everything in this struct is a QString.

KeyMeaning
PackageNameHolds the name of the package.
DescriptionHolds the remote description of the package.
VersionHolds the latest version of the package.
DownloadableArchivesHolds the information on the package data.
SHA1Contains the SHA1 Sum of meta.7z of the remote repo.

This struct is emitted inside a QVector when CheckForUpdates() is finished.
